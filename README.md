## Steps to run - 

1) Clone this repo
2) Set up a postgres database and add it to env variables 
3) Run the following commands
4) ```
   npm install
   nodemon index.js
   ```
5) Go to localhost:3000/books 

## Tech Stack - 

Node.js, Express, EJS, HTML, CSS, JavaScript, PostgresSQL
